# 4kwigs
Turn hair styling into a fully interactive, player-driven criminal or civilian enterprise. 4kwigs is a feature-rich, high-performance roleplay script designed for QB-Core, QBX, and ESX frameworks. It provides a complete economy loop where players purchase raw supplies, deploy modular crafting setups, batch-produce custom wig configurations, and distribute products to dynamically generated street-level clientele.📄 Core Features🛍️ Dynamic Wholesale SupplierSelectable Purchase Quantities: Features a built-in ox_lib input dialog calculation engine allowing players to input exactly how many supplies they want to order.Smart Bounding Vectors: Raycast floor alignment configuration ensuring interaction zones perfectly anchor to the supplier NPC without floating hitbox anomalies.✂️ Modular Crafting MechanicsPrecision Placement Tables: Immersive preview placement system featuring rotation ($\leftarrow$/$\rightarrow$) and custom height controls (PgUp/PgDn). Includes a specialized raycast ground-height engine to completely prevent tables from spawning in mid-air.Batch Production Workflow: Allows players to define an exact crafting quantity. Crafting duration dynamically scales and presents a real-time time preview message before processing starts.📦 Automatic Street Vending (/sellwigs)Dynamic Ped Generation Loop: Spawns random civilian customers sequentially out of the player's direct line of sight, routing them to walk directly up to your exact position.Immersive Hand Props: Automatically spawns and attaches an upright, parallel-aligned shopping bag prop (prop_shopping_bags01) to the character's hand during the sales interaction.Negotiation Engine: Triggers a countdown bargain sequence after proximity arrival. Resolves dynamically into a successful clean/dirty money payout or a temporary rejection animation sequence before cleaning up and spawning the next client.
